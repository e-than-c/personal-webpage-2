FitBook

This is a project that I worked on. It was a lot of fun and I learned a lot from it. I hope you enjoy it!

Overview

FitBook is a fitness app that helps users track their workouts and nutrition. It has a clean and intuitive interface that makes it easy to use. Users can create custom workouts, track their progress, and set goals. The app also provides personalized recommendations based on the user's goals and preferences.
